# Code of Conduct

This project adopts the **[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. The full text is available at the link above and
the canonical source is at https://www.contributor-covenant.org/.

## Scope

The covenant applies to all project spaces, including but not limited to:

- The GitHub repository (issues, pull requests, discussions, wiki, code
  review comments, commit messages).
- Any community channel that explicitly identifies itself as a project
  space (mailing list, chat room, project event).
- Public spaces when an individual is officially representing the project.

## Reporting

Reports of abuse, harassment, or otherwise unacceptable behavior may be
sent to **rock@rockcyber.com**. Reports are confidential and will be
reviewed within 5 business days.

If the report concerns the maintainer, please contact a trusted
intermediary or use a public reporting channel of your choice.

## Enforcement

The maintainer is responsible for enforcement and may take any action
deemed appropriate, ranging from a private warning to a permanent ban
from project spaces. Enforcement actions follow the
[Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
